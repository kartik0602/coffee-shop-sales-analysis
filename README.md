# ☕ Coffee Shop Sales Analysis Dashboard

## 📌 Project Overview

This project presents an interactive Coffee Shop Sales Dashboard built using **Power BI**, **SQL**, and **DAX**. The dashboard helps analyze sales performance, product trends, store performance, and customer purchasing behavior through interactive visualizations.

---

## 🚀 Tools & Technologies

- Power BI
- SQL
- DAX
- Microsoft Excel

---

## 📊 Dashboard Features

### Executive KPIs
- Total Sales
- Total Orders
- Total Quantity Sold
- Month-over-Month (MoM) Growth

### Interactive Visualizations
- Sales Trend Over Time
- Sales by Product Category
- Sales by Product Type
- Sales by Store Location
- Weekday vs Weekend Sales
- Sales by Day & Hour Heatmap
- Calendar Slicer

---

## 📈 Key Business Insights

- Coffee products contribute the highest share of total revenue.
- Morning hours generate the highest number of sales.
- Hell's Kitchen is the top-performing store.
- Weekday sales outperform weekend sales.
- Espresso-based beverages are among the best-selling products.
